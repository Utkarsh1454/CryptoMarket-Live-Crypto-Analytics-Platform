# 🚀 CryptoMarket — Live Crypto Analytics Platform

CryptoMarket is a modern, responsive web-based cryptocurrency dashboard that provides real-time market data, trending insights, and crypto news — all in one place.

---

## 🌐 Live Demo
👉 https://your-live-link.com

---

## 📌 Overview

CryptoMarket allows users to:
- Track live cryptocurrency prices
- Explore detailed coin analytics
- View trending coins and market movers
- Stay updated with latest crypto news

It is designed with a clean UI, fast performance, and real-time API integration.

---

## ✨ Features

### 📊 Market Overview
- Live prices of top cryptocurrencies
- Global stats (market cap, volume, BTC dominance)
- Sort & filter (Top, Gainers, Losers)
- Pagination / Load more system

### 🔍 Smart Search
- Real-time autocomplete
- Search coins by name or symbol
- Keyboard navigation support

### 🪙 Coin Detail Page
- Detailed stats (price, supply, volume)
- Interactive charts (1D, 7D, 1M, etc.)
- Historical data visualization
- ATH / ATL tracking

### 📈 Trending Section
- Top gainers & losers
- High-volume coins
- Trending cryptocurrencies
- Mini sparkline charts

### 📰 Crypto News
- Latest crypto news feed
- Category filtering
- Featured + list view layout

### 💱 Multi-Currency Support
- Supports USD, INR, EUR, BTC, ETH, etc.
- Currency preference saved using localStorage

## 🚀 Getting Started

### 1️⃣ Clone the repository
```bash
git clone https://github.com/your-username/cryptomarket.git
cd cryptomarket
```

### 2️⃣ Run locally
Simply open:
```
index.html
```

Or use **Live Server** for better experience.

---

### 3️⃣ (Optional) Run Backend
```bash
cd backend
pip install -r requirements.txt
uvicorn main:app --reload
```
